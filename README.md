### Key Features

* Processed and cleaned structured loan application data from CSV files, applying binary encoding and feature engineering for model readiness.
* Performed feature selection and multicollinearity analysis, identifying influential predictors such as **CIBIL Score**, **Loan Term**, and **Applicant Income** to improve model generalization.
* Standardized numerical features using **StandardScaler** to ensure consistent model performance across varying input ranges.
* Built a **Logistic Regression** model achieving **91.6% accuracy** on the test dataset for loan approval prediction.
* Developed an interactive **Streamlit** web application with user-friendly inputs such as sliders and dropdown menus for real-time loan eligibility assessment.
* Containerized and deployed the application using **Docker**, enabling reproducible and scalable deployment.
